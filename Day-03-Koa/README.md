1) init npm
    - npm init
2) install koa
    - npm install --save koa
3) install nodemon to restart server after file changes
    - npm install -g nodemon
4) create app.js file
5) run the server 
    - nodemon app.js
6) install koa router module
    - npm install --save koa-router
